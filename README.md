#  CSV Data Backup Plugin (WordPress)

A lightweight and reliable **CSV Data Backup Plugin** for WordPress that allows administrators to **export database records or custom data into CSV files** for backup, reporting, or external processing.

This plugin makes it easy to generate CSV backups of selected data directly from the WordPress admin dashboard without relying on external tools.

---

##  Features

-  Export WordPress data to CSV
-  Select specific data types or tables
-  Secure and user-friendly admin interface
-  Ideal for backups, reporting, and data migration
-  Seamless integration with WordPress

---

##  Tech Stack

| Technology | Purpose |
|------------|---------|
| PHP        | Plugin logic & CSV generation |
| WordPress API | Integration with the admin dashboard |
| HTML       | Admin interface |
| CSS        | Optional styling |
